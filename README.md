jenigrant.com
=============

==Option one:== 
	docker build .
	(currently only works for localhost)

==Option two:==

Make sure nodejs is installed.

To build:
* npm install
* gulp build

To run:
* npm start
* Navigate to http://localhost:3000

-OR-

* Point an apache virtual host to ./public/
* Navigate to your chosen hostname

Voila
